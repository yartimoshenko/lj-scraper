# LiveJournal Archive Scraper

Скрипт для скачивания полного архива блога на LiveJournal — посты, комментарии, картинки — в локальные HTML-файлы с навигацией по годам и тегам.

## Возможности

- Скачивание всех постов за выбранные годы
- Полная загрузка комментариев (из JSON `Site.page`, без UI-взаимодействия)
- Сохранение картинок локально
- Замена внутренних ссылок на локальные файлы
- Генерация навигационного индекса: по годам, тегам, полнотекстовый поиск
- Обход anti-bot защиты через `curl_cffi` с Chrome TLS impersonation
- Async архитектура с единой сессией (как настоящий браузер)
- Автоматический retry при ошибках

## Установка

```bash
pip install curl_cffi lxml
```

## Запуск

```bash
python lj_scraper_scrapling.py
```

Скрипт спросит:
1. URL автора (`https://username.livejournal.com/`) или конкретного поста
2. Папку для сохранения
3. Годы для скачивания (например `2014-2024`)

## Структура результата

```
ArchiveFolder/
├── 01.MAIN.html          # Главная страница с навигацией
└── Content/
    ├── 12345.html         # Отдельные посты
    ├── year_2024.html     # Навигация по годам
    ├── tag_математика.html # Навигация по тегам
    ├── search_index.js    # Индекс для поиска
    └── image.jpg          # Сохранённые картинки
```

## Технические детали

- **HTTP**: `curl_cffi` с Chrome impersonation — единая `AsyncSession` на весь прогон
- **Парсинг HTML**: `lxml` напрямую (не через scrapling) для корректной работы с UTF-8/кириллицей
- **Комментарии**: извлекаются из JavaScript-объекта `Site.page` (JSON), не из HTML
- **Параллелизм**: `asyncio.Semaphore` — контролируемая конкурентность без блокировки
